## 0️⃣ 前提定義（最初に決める）

| 項目 | 内容 |
| --- | --- |
| 対象ユーザー | サークルに入部する前の新入部員 |
| 提供価値 | 気軽に・何度でも・遠慮なくじょぎについて質問できる環境を提供し、新入生の不安を解消＆部員の対応負担を軽減 |
| コア体験 | チャットbotへの質問、じょぎの情報を得る |
| 成功指標 |  ①新入生対応数1.5倍増加 ②対応時間30%削減 ③入部率10%向上  |
